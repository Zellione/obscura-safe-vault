# Suggested Commands

## First-time setup
```bash
scripts/setup.sh          # premake5 + SDL3 + image codecs (calls build_codecs.sh)
scripts/build_codecs.sh   # (re)build just the vendored image codecs into vendor/codecs-prefix
```
Prerequisites: `cmake`, `ninja`, a C++23 compiler, and **nasm** (for libaom / AVIF).

## Generate build files
```bash
scripts/gen.sh            # Ninja (default) — also regenerates compile_commands.json
scripts/gen.sh --gmake    # GNU Make fallback
```

## Build
```bash
scripts/build.sh          # Debug
scripts/build.sh --release
scripts/build.sh --clean  # wipe this config's outputs first, then build (combinable with --release/--gmake)
```

## Run
```bash
build/bin/Debug/osv
build/bin/Release/osv
```

## Tests
```bash
scripts/test.sh           # build + run all tests (Debug)
scripts/test.sh --asan    # AddressSanitizer + UBSan + LSan
scripts/test.sh --release
scripts/test.sh --tsan    # ThreadSanitizer (mutually exclusive with --asan)
```
⚠ `scripts/test.sh` builds ONLY the test binary — for app-side changes,
`scripts/build.sh` must also pass (`-Werror` branch-wide since audit-improvements).
Clang parity check (matches CI's clang leg): `bin/premake5 ninja --cc=clang && ninja -k0 -f build.ninja`,
then restore with `scripts/gen.sh`.

## ASAN-instrumented codecs (audit-improvements)
```bash
scripts/build_codecs.sh --asan   # → vendor/codecs-prefix-asan/ (separate prefix, gitignored)
scripts/test.sh --asan           # premake prefers the asan prefix when it exists
```

## Import benchmark (audit-improvements)
```bash
OSV_BENCH=1 scripts/test.sh                      # N=100 bench (tmpfs)
OSV_BENCH=1 OSV_BENCH_DIR=$HOME/.cache scripts/test.sh   # real-disk numbers
```

## Windows equivalents
```bat
scripts\setup.bat
bin\premake5.exe vs2022
msbuild ObscuraSafeVault.sln /m /p:Configuration=Debug /p:Platform=x64
```

## Packaging
```bash
scripts/package.sh           # Linux tar.gz
makensis /DVERSION=… packaging/windows/osv.nsi   # Windows installer
```

## Serena / LSP
- `compile_commands.json` is regenerated by `scripts/gen.sh` — run after adding/moving files.
- `serena memories check` — check for stale memory references.
